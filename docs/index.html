Who this tool is for

This explorer is designed for PhD and PsyD students applying to APPIC-listed predoctoral internship sites in the 2026–27 match cycle.

The tool lets you filter sites by program type (clinical, counseling, school psychology), agency type, APA accreditation status, stipend range, minimum hour requirements, and more, then view them on a map and build a personal shortlist you can export to CSV.

It is intended as an exploratory aid to help you identify programs worth investigating further, not as a substitute for reading each site's full DCTC or contacting training directors directly.

⚠️

Important: Verify all information before applying

This site displays a static snapshot scraped from the public APPIC online directory on June 1, 2026. Please be aware of the following limitations:

  • Data may be outdated. Training programs update their directory listings throughout the year. Stipends, available positions, eligibility requirements, and accreditation status can change after the snapshot date.
  • Data is self-reported. Information is entered by the training sites themselves. Errors, omissions, and inconsistencies in the original APPIC directory are reflected here.
  • Not all fields are guaranteed accurate. Some sites leave fields blank or report placeholder values. Always read the site's DCTC brochure for authoritative details.
  • This tool is not affiliated with APPIC. It is an independent resource created to make the publicly available data easier to explore. For official match information, use the APPIC website and the AAPI Online portal directly.
